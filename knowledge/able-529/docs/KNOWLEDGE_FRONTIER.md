## Assumptions to challenge every run

1. "Federal conformity means state conformity." (often false)
2. "Plan websites summarize tax law correctly for all taxpayers." (not always; treat as secondary)
3. "A rule true for contributions is also true for rollovers/distributions." (category errors common)
4. "If IRS newsroom says it, implementation details are settled." (can be stale after statutory change)
