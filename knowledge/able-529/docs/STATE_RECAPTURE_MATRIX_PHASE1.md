# State Recapture/Conformity Matrix (Phase 1)

Updated: 2026-03-10

Purpose: build a citation-backed matrix for state treatment of 529->ABLE rollovers.

Legend:
- **Qualified** = likely no recapture/add-back (state treatment aligns with federal pathway).
- **Recapture risk** = likely add-back/recapture exposure.
- **Unclear** = primary authority does not clearly resolve 529->ABLE treatment.

| State | State 529 tax benefit | 529->ABLE state treatment | Primary authority | Confidence | Key unresolved item |
|---|---|---|---|---|---|
| Ohio | Deduction available under ORC 5747.70 | **Recapture risk** (no explicit carveout found for 529->ABLE) | https://codes.ohio.gov/ohio-revised-code/section-5747.70 ; https://tax.ohio.gov/help-center/faqs/income-529-plan-account-deduction/income-529-plan-account-deduction | Medium | Need direct Ohio Taxation statement that 529->ABLE is exempt or not exempt from add-back. |
| Utah | my529 credit/deduction framework | **Likely qualified** if IRC 529 exceptions are met | https://incometax.utah.gov/additions/my529-addback ; https://incometax.utah.gov/credits/my529 | Medium-High | Need explicit Utah Tax Commission language naming 529->ABLE in addback context. |
| Indiana | 529 credit framework | **Likely recapture-triggering** for nonqualified withdrawal pathways; ABLE-specific handling still needs direct in.gov line citation | https://www.in.gov/tos/iesa/tax-credit/tax-credit-faq/ ; https://www.in.gov/dor/i-am-a/individual/tax-credits/ | Medium | Capture direct Indiana code or DOR line explicitly addressing 529->ABLE recapture treatment. |
| Minnesota | Credit/subtraction via M1529 framework | **Unclear** for 529->ABLE specifically; recapture mechanics clearly exist for nonqualified distributions | https://www.revenue.state.mn.us/recapture-tax ; https://www.revenue.state.mn.us/sites/default/files/2025-10/m1529-25-grid.pdf ; https://www.revisor.mn.gov/statutes/cite/290.0684 | Medium-Low | Determine whether MN treats 529->ABLE as a qualified rollover or recapture event. |

## Notes

1. This matrix is a planning research artifact, not filing advice.
2. Federal permanence of 529->ABLE does not force state conformity.
3. For low-confidence states, form-line instructions and state conformity dates are required before "advisor-safe" status.
