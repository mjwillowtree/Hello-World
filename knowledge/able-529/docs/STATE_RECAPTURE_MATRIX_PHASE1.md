## Immediate promotion criteria (to call a state "filing-grade")

1. Statute/regulation citation with line-level wording.
2. Current-year form/instruction line and reporting treatment.
3. Conformity-year check (rolling vs static).
4. Explicit uncertainty note if authority is silent.
