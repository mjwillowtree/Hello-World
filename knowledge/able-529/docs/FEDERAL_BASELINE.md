# Federal Baseline (ABLE + 529 Interaction)

Updated: 2026-03-10

## Confirmed baseline rules

1. **Eligibility expansion in 2026**
   - Disability onset test shifts to before age 46 (from before age 26), effective 2026-01-01.
   - Sources:
     - SECURE 2.0 statute: https://www.congress.gov/117/plaws/publ328/PLAW-117publ328.pdf
     - SSA POMS: https://secure.ssa.gov/POMS.NSF/lnx/0501130740

2. **SSI resource treatment**
   - ABLE balances are excluded for SSI resource purposes up to $100,000.
   - Excess above $100,000 can suspend SSI cash payments when countable resources exceed thresholds.
   - Sources:
     - https://www.ssa.gov/ssi/spotlights/spot-able.html
     - https://secure.ssa.gov/POMS.NSF/lnx/0501130740

3. **529->ABLE rollover legality after 2025 sunset concern**
   - Current IRC 529 text now shows ABLE rollover language without the old "before January 1, 2026" limitation.
   - U.S. Code amendment notes identify the change as Pub. L. 119-21, section 70117(a), and effective date language indicates applicability to taxable years beginning after December 31, 2025.
   - Practical conclusion: federal 529->ABLE rollover treatment is no longer a pre-2026-only rule.
   - Sources:
     - Current IRC 529 text + amendment/effective date notes: https://uscode.house.gov/view.xhtml?req=(title:26%20section:529%20edition:prelim)

4. **2026 ABLE base annual contribution interpretation**
   - Best current legal reading is **$20,000** for 2026.
   - Reason: §529A(b)(2)(B)(i) applies ABLE-specific indexing language substituting "1996" for "1997" when applying §2503(b)(2)(B).
   - Sources:
     - https://uscode.house.gov/view.xhtml?req=granuleid:USC-prelim-title26-section529A&num=0&edition=prelim
     - https://uscode.house.gov/quicksearch/get.plx?section=2503&title=26
     - https://www.irs.gov/newsroom/irs-releases-tax-inflation-adjustments-for-tax-year-2026-including-amendments-from-the-one-big-beautiful-bill

5. **ABLE-to-Work additional contribution mechanics (core)**
   - Additional contribution cap is the lesser of compensation includible in gross income or one-person poverty line amount for the prior calendar year.
   - Retirement-plan disqualifier is contribution-based and explicitly references contributions to defined contribution plans meeting 401(a)/403(a), section 403(b), and section 457(b) arrangements.
   - IRS 1099-QA/5498-QA instructions emphasize that the poverty guideline uses the beneficiary's state of residence and that the beneficiary must maintain records supporting eligibility.
   - Sources:
     - IRC 529A(b)(2)(B)(ii), 529A(b)(7): https://uscode.house.gov/view.xhtml?req=(title:26%20section:529A%20edition:prelim)
     - TD 9923 / final regulations discussion in IRB (including ABLE-to-Work recap): https://www.irs.gov/irb/2020-50_IRB#TD-9923
     - IRS instructions (2025): https://www.irs.gov/instructions/i1099qa

## Authority hierarchy (advisor-safe conflict rule)

When sources conflict, rely in this order:
1. Enacted statute and explicit effective-date language.
2. Current codified text plus amendment/effective-date notes (U.S. Code editors' notes are highly useful but not themselves enacted law).
3. Final regulations / Treasury decisions / notices in IRB.
4. IRS forms, instructions, publications, newsroom summaries.
5. Program marketing pages and third-party commentary.

Conservative rule: if lower-tier guidance appears stale after statutory change, default to statute plus effective-date language and disclose implementation lag risk.
