# Authority Ladder for ABLE/529 Answers

Updated: 2026-03-09

Purpose: provide practitioner-safe conflict resolution when sources disagree.

## Ranking (highest to lowest)

1. **Statute and codified amendments** (IRC text + amendment history)
2. **Treasury regulations / final regs**
3. **IRS forms and instructions**
4. **IRS publications / IRS newsroom content**
5. **State agency guidance and plan administrator materials** (for state/program operations)
6. **Secondary summaries** (advisor articles, blogs)

## Conflict-resolution rule

If lower-tier guidance conflicts with higher-tier authority:

- Treat higher-tier authority as controlling.
- Explicitly disclose the conflict in written advice.
- Preserve a citation chain in client file notes.

## Currency-check protocol (mandatory)

Before relying on legacy summaries:

1. Check whether the controlling code section has later amendments.
2. Confirm whether IRS forms/instructions were updated for the relevant tax year.
3. Verify state conformity date and state-specific recapture instructions.
4. Mark any unresolved mismatch as a risk flag, not as settled law.

## Example from current knowledge base

- Topic: 529->ABLE rollover timing.
- Risk: older IRS newsroom language referenced pre-2026 limit.
- Control: current code amendment history shows Congress removed the "before January 1, 2026" phrase in 2025.
- Practice: cite current code text/amendment notes first, then contextualize legacy pages as historical.
